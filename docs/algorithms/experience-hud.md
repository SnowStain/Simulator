# 经验、等级与左下角 HUD

## 目标

左下角 HUD 不再只是文本状态栏，而是直接展示当前机器人的核心战斗状态：

- 机器人身份与剪影
- 当前等级与经验进度
- 当前血量和血量上限
- 缓冲能量和当前功率上限
- 当前增益/减益及倍率

这块 HUD 的原则是：只展示局内决策最需要的信息，减少和 `F5` 按键指南、右侧提示栏的重复。

## 经验阈值

机器人等级阈值按规则资料中的 10 级体系接入：

```text
Lv1 = 0
Lv2 = 550
Lv3 = 1100
Lv4 = 1650
Lv5 = 2200
Lv6 = 2750
Lv7 = 3300
Lv8 = 3850
Lv9 = 4400
Lv10 = 5000
```

HUD 的经验弧使用当前等级阈值和下一等级阈值计算：

```text
ratio = (experience - threshold(level)) / (threshold(level + 1) - threshold(level))
```

显示角度为 `2π/3`，也就是圆周的三分之二。

## 能量机关经验

能量机关命中和激活会给经验：

- 单次命中：射手机器人获得 `ringScore * 20` 经验。
- 小能量机关激活：己方存活可升级机器人分摊 `1200` 经验。
- 大能量机关激活：己方存活可升级机器人分摊 `750` 经验。

当前可升级机器人排除了工程机器人，避免工程因机制定位不同被错误纳入战斗升级链路。

## HUD 结构

左下角新面板分成三块：

- 左侧圆形头像：显示机器人剪影、队伍色外圈和等级数字。
- 圆形左侧经验弧：用金色进度弧展示当前等级经验进度。
- 右侧状态区：上方 buff 图标，中部梯形血条，下方缓冲能量条。

血条使用队伍颜色填充，文本为：

```text
currentHp / maxHp
```

缓冲能量条显示灰色底和浅色填充，右侧显示当前功率上限：

```text
powerLimit W
```

## Buff 图标

Buff 图标统一使用小金黄色圆形底和外圈进度条：

- 防御增益：护盾
- 易伤：破碎护盾
- 攻击增益：剑
- 热量冷却增益：时钟
- 回血增益：十字
- 缓冲能量增益：电池

如果 buff 文本中包含倍率，例如 `x1.25`，会显示在图标右下角。

## 2026-04-26：固定侧面头像

左下角圆形头像现在使用固定机器人侧面剪影，而不是实时车体俯视图。这样 HUD 中的头像不会随着车体 yaw 旋转，和选车/外观预览中的侧视认知保持一致。

Buff 图标行整体上移，倍率数值使用半透明底显示，避免数值被下方血条遮挡。

## 后续维护要求

新增 buff、经验来源或等级规则时，需要同步修改：

- `ArenaInteractionService` 中的经验发放逻辑
- `RuleSet` 中的等级上限和属性表
- `Simulator3dForm` 中的 HUD 图标和倍率显示
- 本文档和项目日志
